# Music Player App

A responsive music player built with HTML and CSS and vanilla JS. Features custom audio controls and a sleek UI.

console.log("Wrote each and every line myself(no-AI)! hehe")

## Features
- Play/pause/skip controls
- Responsive layout
- Stylish design
- Black themed

## How to Use
- Download or clone the repo.
- Add mp3 files to music-player/songs/
- Open `index.html` in your browser and enjoy!
